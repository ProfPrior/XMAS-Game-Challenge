1) Add Additional Animations (Advanced+)

The following animations are missing from the project:

  • The character looks forward and walks to the right
  • The character looks forward/right and idles
  • The character looks backward and walks to the left
  • The character looks backward/left and idles

There are TODO comments in the PlayerMovement script to help guide you through the solution. Please ensure that the new animation clips have the same FPS as the existing clips.

Tip 1: Animator

Inspect the Animator and examine the clips already in use. In the scene, you‚Äôll find a prefab named "Character".

Tip 2: FPS

If the FPS for an animation clip is greyed out in the Inspector, you can change it by switching to "Debug" mode.

Tip 3: Reusing Animations

You can copy the keyframes from an existing animation (e.g., walking right) into a new clip. To mirror the animation, add the Flip X property on the SpriteRenderer component and toggle it to flip the sprite.

2) Add (Christmas) Background Music (Beginner)

Add a background music track to the scene that plays continuously during the game.

Tip 1: Audio Source

You need an AudioSource component to play an audio clip. Create an empty GameObject, add an AudioSource component to it, and select an audio clip in the AudioSource settings.

Tip 2: Settings

Ensure the AudioSource is set to 2D (so it‚Äôs audible everywhere) and enable the "Loop" option so the music repeats automatically.

3) Create Collectible Prefabs (Advanced)

Create a collectible prefab, such as a gift, that the player can pick up by walking over it. Make sure the collectible can be reused multiple times throughout the scene.

Tip 1: Collider

The gift prefab needs a Collider component. Set the collider to "Trigger" mode to detect when the player interacts with it.

Tip 2: Collecting in Code

Use the OnTriggerEnter method to detect when the player interacts with the collectible. Check the tag of the other object using if (other.CompareTag("Player")) to ensure only the player can collect it.

4) Add a Pickup Sound Effect (Beginner)

Play a sound effect whenever the player collects a gift.

Tip 1: AudioClip

You can use an AudioSource component to play a short sound effect. Add an AudioSource to the gift prefab and set it to play the sound when the player collects the item.

Tip 2: Trigger Sound in Code

Call AudioSource.Play() in the OnTriggerEnter method to play the sound when the gift is collected.

5) Create a UI for Collected Gifts (Advanced)

Display a counter in the UI that shows how many gifts the player has collected. The counter should update automatically as more gifts are collected.

Tip 1: Canvas

Create a Canvas in the scene and add a Text or TMP_Text element to display the count of collected gifts.

Tip 2: Update UI

In your script, use Text.text = collectedGifts.ToString() or similar to update the counter dynamically whenever a gift is collected.
